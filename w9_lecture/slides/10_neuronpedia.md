# Neuronpedia: Interactive Feature Explorer

## What is Neuronpedia?
- An **open-source, interactive platform** for exploring SAE features across different models
- Created by Johnny Lin — available at **neuronpedia.org**
- Supports **Gemma-2-2B, Gemma 3 (270M–27B), Llama 3.1/3.2, Qwen3-4B, GPT-2** and more

## Core capabilities:

**1. Feature browsing**
- Top activating examples, activation patterns, auto-generated descriptions, logit effects
- Over **1 million features** catalogued across models

**2. Attribution graphs (circuit tracing)**
- Interactive visualizations tracing how a model arrives at an output
- Nodes = features, edges = connections between them — reveals multi-hop reasoning circuits

**3. Model steering**
- Clamp or amplify SAE features to modify model behavior in real time
- The generalized version of **"Golden Gate Claude"** — works with any feature on supported models

**4. Gemma Scope 2**
- Google DeepMind released SAEs + transcoders for the entire **Gemma 3 family**, all browsable on Neuronpedia

## Why it matters:
- Makes mechanistic interpretability **accessible** — no coding required to explore
- Enables **community-driven** verification and annotation of features
- A practical tool for auditing model internals
